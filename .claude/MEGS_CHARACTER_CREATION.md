# MEGS Character Creation System

## Overview

The Mayfair Exponential Gaming System (MEGS) uses a point-buy character creation system where every Attribute, Power, Skill, Advantage, and Wealth AP must be purchased using Hero Points.

**Starting Budget: 450 Hero Points**

---

## Core Concepts

### Factor Cost (FC)

Everything measured in APs has a **Factor Cost** representing its game utility.

- **Range**: 1 to 10
- **Higher FC = More expensive** per AP
- Used with the **AP Purchase Chart** to determine Hero Point cost

## Powers & Skills

### Cost Formula

**Total Cost = Base Cost + AP Cost (from chart)**

Where:
- **Base Cost** = One-time cost for acquiring the ability at 0 APs
- **AP Cost** = Looked up on AP Purchase Chart using FC and desired APs

### Example: Energy Blast

- Factor Cost: **3**
- Base Cost: **15 HP**
- Desired APs: **8**

**Calculation:**
1. Look up FC3, 8 APs on chart = **30 HP**
2. Add Base Cost: 30 + 15 = **45 HP total**

### Example: Thief Skill

- Factor Cost: **8**
- Base Cost: **10 HP**
- Desired APs: **4**

**Calculation:**
1. Look up FC8, 4 APs on chart = **24 HP**
2. Add Base Cost: 24 + 10 = **34 HP total**

## Linking Powers & Skills

A Power or Skill may be **linked** to its associated Attribute, meaning:

### Linking Rules

1. **Must buy APs equal to the linked Attribute's APs**
   - If STR is 5 APs, linked Energy Blast must also be 5 APs

2. **Reduce Factor Cost by 2** (minimum FC 1)
   - Energy Blast FC 3 → FC 1 when linked

3. **If Attribute drops, linked ability drops too**
   - Temporary or permanent attribute loss affects linked powers

4. **Harder to raise later**
   - Must raise Attribute AND all linked abilities simultaneously

### Linking Example: Energy Blast Linked to STR

**Unlinked:**
- STR 5 = 24 HP (FC6, 5 APs)
- Energy Blast 8 (FC3, Base 15) = 45 HP
- **Total: 69 HP**

**Linked to STR 5:**
- STR 5 = 24 HP (FC6, 5 APs)
- Energy Blast 5 (FC1, Base 15) = 5 + 15 = 20 HP
- **Total: 44 HP**
- **Savings: 25 HP**

**Trade-off:** Energy Blast is now capped at 5 APs and tied to STR

## Bonuses & Limitations

### Bonuses (Increase FC)

Add to Factor Cost to represent enhanced utility.

**Examples:**
- **Usable on Others**: +5 FC
- **Area Effect**: +2 FC
- **Multi-Attack**: +1 FC

### Limitations (Decrease FC)

Subtract from Factor Cost (minimum FC 1).

**Examples:**
- **No Range**: -1 FC
- **Concentration Required**: -1 FC
- **Limited Uses**: -1 to -3 FC

### High Factor Costs (FC > 10)

When modified FC exceeds 10:
1. Calculate cost at **FC 10**
2. Add cost at **FC (excess)**

**Example: FC 15**
- FC 15 = Cost at FC 10 + Cost at FC 5
- 10 APs at FC 15 = (10 APs at FC 10) + (10 APs at FC 5)
- = 160 + 80 = **240 HP**

## Drawbacks

When character costs exceed 450 HP budget, players may take **Drawbacks** to offset the excess.

### Common Drawbacks

- **Physical Restriction**: 10-30 HP
- **Psychological Instability**: 10-30 HP
- **Secret Identity**: 10-15 HP
- **Dependent**: 5-20 HP
- **Enemy**: 5-30 HP (based on power level)

**Drawbacks add Hero Points back to the budget** but create story complications.

## Character Creation Workflow

### Step 1: Purchase Attributes (298 HP budget guideline)

Calculate each attribute's cost using FC and AP Purchase Chart.

### Step 2: Purchase Powers

For each power:
1. Determine Factor Cost (from power description)
2. Determine desired APs
3. Decide if linking to an attribute
4. Apply bonuses/limitations (adjust FC)
5. Look up AP cost on chart
6. Add Base Cost
7. Total = Base Cost + AP Cost

### Step 3: Purchase Skills

Same process as powers:
1. Determine FC and Base Cost
2. Choose APs
3. Consider linking
4. Calculate using chart + Base Cost

### Step 4: Purchase Advantages

Add fixed costs for each advantage.

### Step 5: Purchase Wealth

Use FC2 to determine cost based on desired income level.

### Step 6: Calculate Total

Add all costs:
- All 9 Attributes
- All Powers
- All Skills
- All Advantages
- Wealth

### Step 7: Balance Budget

- **Under 450 HP**: You have points remaining
- **Over 450 HP**: Must take Drawbacks to offset excess
- **Exactly 450 HP**: Perfect balance

## Implementation Notes for Foundry VTT

### Current Issues in Codebase

1. **Wrong Cost Formula**
   - Current: `totalCost = baseCost + (factorCost × APs)`
   - Correct: `totalCost = baseCost + apCostChart[APs][FC]`

2. **Missing AP Purchase Chart**
   - Need to add chart as data structure in config or JSON

3. **No Budget Tracking**
   - Need to track total HP spent vs. 450 HP budget
   - Display in character sheet with visual indicators

4. **Linking Not Implemented**
   - Need to reduce FC by 2 when power/skill is linked
   - Enforce AP cap matching linked attribute
   - Visual indicator for linked abilities

5. **Drawbacks Don't Affect Budget**
   - Drawbacks should add HP back to available pool
   - Need to calculate net budget: 450 + drawbacks - spent

### Recommended Implementation

1. Create `assets/data/apCostChart.json` with the purchase chart
2. Update `module/helpers/config.mjs` to load and expose chart
3. Refactor `module/documents/item.mjs` cost calculation
4. Add `heroPointBudget` tracking to `module/documents/actor.mjs`
5. Update character sheet templates to display budget
6. Add linking logic with FC reduction
7. Create character builder UI for budget-aware creation
